# Skills de Valor

Sitio web estático para la marca personal "Skills de Valor" listo para despliegue.

## Estructura del proyecto
- `public/index.html`: página principal con secciones de Inicio, Conócenos, Apóyanos, Contáctanos y Linktree.
- `public/styles.css`: estilos en tonos pasteles claros y paleta de colores de marca.
- `public/script.js`: navegación suave entre secciones.
- `public/assets/logo-skills-de-valor.png`: logotipo renombrado y listo para producción.

## Cómo usar
1. Abre `public/index.html` en tu navegador.
2. Reemplaza los enlaces de la sección `Linktree` con tus perfiles reales.
3. Para desplegar, usa un hosting estático y apunta al contenido de la carpeta `public`.

## Despliegue recomendado
- GitHub Pages: configura la publicación desde la carpeta `public` o usa la rama `gh-pages`.
- Netlify: arrastra la carpeta `public` al panel de sitios o configura la carpeta de publicación a `public`.
- Vercel: selecciona despliegue estático y ajusta la carpeta raíz a `public`.

Si prefieres, puedo ayudarte a crear un flujo de despliegue con GitHub Actions.
